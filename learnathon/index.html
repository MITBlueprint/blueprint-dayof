Track Descriptions

Web Development

The web development track introduces you to HTML, CSS, and JavaScript and helps you create your very own personal website. We’ll be starting off with HTML and CSS, teaching fundamentals of programming with JavaScript, and then introducing Meteor, a web development framework that makes it easy to build realtime web apps fast.

iOS Development

The iOS track explores the basics of iOS game development. Make School, a company that aims to empower students to build and ship products, will be teaching classes on mobile development practices and iPhone game creation in Swift.

Android Development

The Android track consists of classes that teach the essentials of software development in Java and introductory practices of Android development. Workshops will cover basic Java programming and walk you through development of your first Android app!

Mentorship

Blueprint is a learning event, so we’ve invited MIT students and industry professionals to guide you throughout the weekend.

You’ve been matched with a mentor group, which you can find by logging in at blueprint.hackmit.org, and you’ll be checking in with this group at specified times throughout the day. Your mentor will clarify confusing concepts, troubleshoot bugs, and share experiences about computer science in college and as a career path. Don’t be afraid to ask questions—your mentors are here to help you!
